# 🏠 Real Estate Market Trends Dashboard - Power BI

## 📌 Overview
This project is a Power BI dashboard that analyzes real estate market trends using property pricing, rental yields, supply-demand metrics, and economic indicators.

The dashboard provides interactive visualizations to help investors, developers, and analysts make informed real estate decisions.

---

## 🎯 Objectives
- Analyze property prices and rental yields.
- Identify high-demand and high-growth areas.
- Track market demand and supply conditions.
- Visualize geographical hotspots using maps.
- Support investment and development decisions with data insights.

---

## 📊 Features
### 1. Property Price Analysis
- Average property prices by city/region
- Year-over-year growth trends
- Price comparison by property type

### 2. Rental Yield Insights
- Rental income analysis
- ROI comparison across locations
- Occupancy rate tracking

### 3. Market Demand & Supply
- Available listings vs sold properties
- Demand forecasting
- Market saturation analysis

### 4. Geographic Visualizations
- Heat maps for property hotspots
- Regional trend analysis
- Interactive map filters

### 5. KPI Metrics
- Average Property Price
- Rental Yield %
- Occupancy Rate
- Total Listings
- ROI %

---
